How do I create a new project?

Click the "New Project" button in the sidebar or use the Quick Actions panel. Enter the project name, description, and choose a color. Projects help organize related tasks together.

Can I import tasks from other apps?

Yes! Go to Settings > Data > Import Data. You can import tasks from CSV or JSON files. Check our documentation for supported formats and templates.

How do I share tasks with team members?

Currently, Taskly is designed for individual use. Team collaboration features are planned for future releases. You can export tasks and share the file with others.

Is my data backed up automatically?

Taskly stores all data locally in your browser. We recommend regularly exporting your data via Settings > Data > Export Data. Cloud backup features are coming soon.

How do I change the theme?

Go to Settings > Preferences. You can choose between Light, Dark, or Auto (system) themes. Changes apply immediately.
