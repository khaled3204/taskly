Taskly Documentation

Learn how to use Taskly effectively with this comprehensive guide

Introduction

Taskly is a professional task management application designed to help individuals and teams organize their work efficiently. With a clean interface and powerful features, Taskly makes it easy to manage tasks, projects, and deadlines.

Key Features

  • Task creation with priorities, due dates, and labels
  • Project organization for grouping related tasks
  • Interactive calendar view for planning
  • Comprehensive dashboard with analytics
  • Multi-language support (English & Turkish)
  • Dark/Light theme switching
  • Local data storage with import/export capabilities

Note: Taskly stores all data locally in your browser. No data is sent to external servers.

Installation

Taskly is a web-based application that runs in your browser. No installation is required!

Browser Requirements

Taskly works best with modern browsers that support:

  • HTML5 Local Storage API
  • CSS Grid & Flexbox
  • ES6 JavaScript Features

Quick Start Guide

Get started with Taskly in just a few minutes:

Step 1: Create an Account

Click "Sign Up" on the login page and enter your information. Your data will be stored securely in your browser.

Step 2: Add Your First Task

  1. Click "Quick Add" in the sidebar or "Add Task" in any view
  2. Enter a task title (required)
  3. Add optional details like description, due date, priority
  4. Click "Add Task" to save

Step 3: Organize with Projects

Create projects to group related tasks together:

1. Click "New Project" in sidebar 2. Enter project name and description 3. Choose a color for visual identification 4. Click "Create Project"

Tip: When you first sign up, Taskly creates sample tasks and projects to help you get started.

Tasks Management

Tasks are the core building blocks of Taskly. Here's how to manage them effectively:

Task Creation

When creating a task, you can specify:

  • Title: Brief description of the task
  • Description: Detailed information (optional)
  • Priority: Low, Medium, or High
  • Due Date: When the task should be completed
  • Project: Which project this task belongs to
  • Labels: Comma-separated tags for organization

Task Actions

Complete Tasks

Click the checkbox next to any task to mark it as completed. Completed tasks appear in the "Completed" view.

Edit Tasks

Click the edit icon (pencil) on any task to modify its title.

Delete Tasks

Click the delete icon (trash) to remove a task. You'll be asked to confirm before deletion.

Projects

Projects help you organize related tasks. Here's what you need to know:

Creating Projects

  1. Click "New Project" in the sidebar
  2. Enter a descriptive name
  3. Add an optional description
  4. Choose a color for visual identification
  5. Click "Create Project"

Project Management

Each project has:

  • Task count badge showing number of tasks
  • Color-coded identification in sidebar
  • Dedicated project view showing all related tasks
  • Ability to delete projects (tasks become unassigned)

Warning: When you delete a project, its tasks are not deleted but become unassigned (no project).

Data Import & Export

Taskly provides robust data management features:

Exporting Data

  1. Go to Settings → Data
  2. Click "Export Data"
  3. A JSON file will be downloaded
  4. This file contains all your tasks, projects, and settings

Importing Data

  1. Go to Settings → Data
  2. Click "Import Data"
  3. Select a previously exported JSON file
  4. Confirm to replace current data

Important: Regularly export your data as a backup. Taskly stores data locally, so clearing browser data will remove your tasks.

Clearing All Data

To start fresh or troubleshoot issues:

1. Go to Settings → Data 2. Click "Clear All Data" 3. Confirm the action 4. All tasks, projects, and settings will be removed
